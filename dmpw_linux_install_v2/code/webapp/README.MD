# VN Army DMP Webapp

## Available Scripts

### Running the development server.

```bash
    npm run dev
```

### Building for production.

```bash
    npm run build
```

### Running the production server.

```bash
    npm run start
```

## Deploy docker
```bash
    # Build the Docker image
    docker build -t vn-army-dmp-webapp .
    
    # Run the container
    docker compose up -d
```
